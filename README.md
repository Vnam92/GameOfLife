#Conway's "Game Of Life" 
Example MVC project with unit-tests.
##Build project
`npm build` for development purposes. Serves a webpack app.
##Run dev server
`npm start` for serving a webpack bundle.
##Testing
`npm test` for run Karma testing.
##Features
+ Javascript 
+ Webpack 2.x
+ Jasmine 2.5.x
+ Karma 1.4.x
